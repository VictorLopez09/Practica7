El frontend.

Se refiere a la parte visible de un sitio web o aplicación, es decir, la interfaz que los usuarios ven y con la que interactúan. La función del frontend es presentar la información de manera clara y atractiva, y permitir a los usuarios interactuar con la aplicación de manera intuitiva.

Los funadamentos son:


  • HTML: Es el lenguaje de marcado que se utiliza para estructurar el contenido de una página web. Con HTML se pueden crear encabezados, párrafos, tablas, formularios y otros elementos que conforman la estructura de una página web.

  • CSS: Es el lenguaje que se utiliza para definir el estilo y la presentación visual de una página web. Con CSS se pueden controlar los colores, fuentes, tamaños, diseños y otros aspectos visuales de una página web.

  • JavaScript: Es el lenguaje de programación que se utiliza para crear interacciones dinámicas en una página web. Con JavaScript se pueden crear efectos visuales, animaciones, validar formularios y otras funcionalidades que mejoran la experiencia de usuario.
